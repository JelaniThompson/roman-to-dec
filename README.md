# roman-to-dec
> Convert a roman numeral to a base-10 integer.

## Installation
```shell
$ npm install --save jamen/roman-to-dec
```

## Usage
```javascript
var rtd = require('roman-to-dec');

exports('MDCC');
// => 1700
```
